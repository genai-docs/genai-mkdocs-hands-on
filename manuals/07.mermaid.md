# Mermaid記法

Mermaid記法を利用したMarkdownファイルをコピーしVS Codeのプレビューで確認。

```bash
cp -rf reference/07.mermaid/. docs/
```

ルートのmkdocs.ymlを開き、pymdownx.superfencesに設定を追加します。

pymdownx.superfencesnの後ろの「:」がある点に注意してください。

```yml
markdown_extensions:
  - admonition
  - pymdownx.details
  - pymdownx.superfences:
      custom_fences:
        - name: mermaid
          class: mermaid
          format: !!python/name:pymdownx.superfences.fence_code_format
```

MkDocsを起動する

```bash
nr mkdocs
```
